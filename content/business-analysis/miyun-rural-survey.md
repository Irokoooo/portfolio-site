# 北京密云产销合作调研

## 项目背景（Context）

密云区作为北京市重要的农业生态屏障，近年来积极推进农产品产销对接模式创新。本次调研受区农业农村局委托，深入 12 个行政村，系统梳理产销链条中的断点与瓶颈。

**调研周期：** 2025.06 — 2025.08（10 周）

**调研方式：** 半结构化访谈 + 问卷调查 + 参与式观察

## 调研设计

### 样本框架

| 调研对象 | 数量 | 方法 |
|----------|------|------|
| 农户 | 86 户 | 入户访谈 + 问卷 |
| 收购商/经纪人 | 23 家 | 深度访谈 |
| 电商平台运营者 | 8 家 | 半结构化访谈 |
| 区级政府部门 | 4 个 | 政策文件分析 |

### 核心发现

经过 Python 数据清洗与可视化分析，识别出产销链条中的三大断点：

1. **信息不对称**：农户获取市场价格信息滞后平均 **3.2 天**，导致被动定价
2. **冷链缺口**：全区冷链覆盖率仅 **34%**，夏季损耗率高达 18%
3. **品牌空白**：89% 的受访农户无自有品牌，产品溢价能力极弱

## Python 可视化成果

使用 `pandas` + `matplotlib` + `folium` 构建了三类图表：

- **产销地图**：可视化农产品从田间到市场的流转路径，标注各环节加价率
- **价格波动热力图**：展示 2020–2025 年主要品类（板栗、蜂蜜、鲜鱼）的季节性价格波动
- **供应商网络图**：用 `networkx` 绘制收购商之间的合作关系网络

## 策略报告（5000 字主笔）

报告提出"三步走"数字化推广策略：

### 第一步：信息基础设施建设（0–6 个月）
- 在 12 个村委会部署价格信息公示屏
- 接入北京农产品大数据平台实时价格 API

### 第二步：品牌孵化（6–18 个月）
- 遴选 5–8 个单品打造"密云原产地"区域品牌
- 与抖音、拼多多签订直播带货框架协议

### 第三步：冷链网络整合（18–36 个月）
- 以 PPP 模式引入社会资本补建冷链节点
- 目标：冷链覆盖率提升至 **70%**，损耗率压降至 **8%** 以下

## 项目成果

本报告已提交区农业农村局，并在 2025 年密云区乡村振兴工作推进会上作为典型案例报告。
